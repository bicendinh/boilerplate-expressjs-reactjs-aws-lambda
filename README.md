### Quick Start
* Install serverless

```shell
npm install -g serverless
```

* Install Packages & Build Front-end

```shell
yarn install # install serverless package
cd frontend 
yarn install && yarn run build # build front end
cd ..
```
* Deploy to AWS 

```shell
sls deploy
```
